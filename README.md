# s37syed-website  
Repository for my website's development

## Software used  
* AngularJS2 (via TypeScript)  
* HTML5  
* JavaScript  
* CSS3  
* Bootstrap 3  
* jQuery  
* Less  
* NodeJS  

## Build/Test  
##### Install dependencies for website  
`$ npm install`  

##### Installing Bootstrap 3  
`$ bower install`  

##### Running Web Server  
`npm start`  

##### LESS CSS compiling (make sure to do npm install in bower_components)
`grunt watch`
