# s37syed-website  
Repository for my website's development

## Software used  
* AngularJS 5 (via TypeScript)  
* HTML5  
* JavaScript  
* CSS3  
* Bootstrap 4  
* jQuery 
* NodeJS  

## Build/Test  
##### Install dependencies for website  
`$ npm install`  

##### Running Local Dev Server  
`ng serve`   

##### Build Prod
`ng build --prod`
